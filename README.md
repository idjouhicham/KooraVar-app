# 🏆 كووورة Pro — دليل الإعداد الكامل

## ما تم بناؤه
- **index.html** — تطبيق PWA كامل (مباريات، ترتيب، لاعبون، بث، إدارة)
- **manifest.json** — إعدادات PWA لتثبيت التطبيق على الجوال
- **sw.js** — Service Worker ذكي بـ 3 استراتيجيات تخزين

---

## الخطوة 1 — ربط TheSportsDB API

### المفتاح المجاني (يكفي للبداية)
المفتاح المجاني هو الرقم `3` وهو مدمج في التطبيق.

### إذا اشتريت مفتاحاً مدفوعاً:
1. افتح التطبيق
2. اضغط **⚙️ الإدارة**
3. أدخل مفتاحك في خانة **مفتاح TheSportsDB API**
4. اضغط **حفظ المفتاح**

### ما يجلبه TheSportsDB تلقائياً:
| البيانات | المصدر |
|----------|--------|
| مباريات البريميرليج | ✅ تلقائي |
| مباريات لاليغا | ✅ تلقائي |
| مباريات دوري روشن السعودي | ✅ تلقائي |
| مباريات بوندسليغا | ✅ تلقائي |
| ترتيب الدوريات | ✅ تلقائي |
| شعارات الفرق | ✅ تلقائي |
| تفاصيل المباراة | ✅ عند الضغط عليها |

## الخطوة 2 — رفع الملفات على GitHub Pages

```bash
# 1. أنشئ repository جديد على github.com
# 2. ارفع ملفات frontend/ كلها
# 3. Settings → Pages → Source: main branch → /root
# 4. التطبيق سيعمل على: https://اسمك.github.io/اسم-الريبو
```

**ملفات ترفعها:**
```
index.html    ← الملف الرئيسي
manifest.json ← إعدادات PWA  
sw.js         ← Service Worker
icon.svg      ← الأيقونة (أو icon-192.png / icon-512.png)
```

---

## الخطوة 3 — ربط Backend على InfinityFree

1. افتح التطبيق → **⚙️ الإدارة**
2. في خانة **رابط Backend** أدخل:
   ```
   https://موقعك.infinityfreeapp.com
   ```
3. اضغط **حفظ الرابط**

---

## الخطوة 4 — تثبيت التطبيق كـ Native App

### على Android:
1. افتح الرابط في Chrome
2. ستظهر رسالة **"إضافة إلى الشاشة الرئيسية"**
3. اضغط **إضافة** ✅

### على iPhone:
1. افتح الرابط في Safari
2. اضغط زر **المشاركة** (الأسهم)
3. اختر **"إضافة إلى الشاشة الرئيسية"** ✅

## استكشاف الأخطاء

| المشكلة | الحل |
|---------|------|
| لا تظهر مباريات | تحقق من رابط Backend في الإدارة |
| شعارات الفرق لا تظهر | تحقق من اتصال الإنترنت |
| خطأ CORS | تأكد من وجود `.htaccess` على InfinityFree |
| التطبيق لا يُثبَّت | تأكد من أن الموقع HTTPS |
